## Project Overview

**SURF (Smart Uncovering Reaction Feedback)**
> 비대면 학습 솔루션 - 강의자와 학습자의 상호작용을 바탕으로 한 분석 시스템

## Features (ver.1.0.0)
### **[교수자]**
- 집중도 데이터: 집중도 추이 그래프 제공
- 수업 방식 구간별 자동 태깅
- 수업 방식 추천 및 피드백
### **[학생]**
  - 웹캠 데이터 활용 집중도 분석
  - 수업 중 실시간 필기 및 피드백 


## Getting Started
1. Clone
```
git clone https://github.com/FacerAin/2020-SW-Festival-PSY.git
```

2. Start Node Server
```
cd psy-page
npm install
npm start
```

3. Build
```
cd psy-page
npm run build --prod
```
